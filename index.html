Windows 右键菜单快速验证文件哈希

发表于 2025-05-10 分类于 Windows 本文字数： 640 阅读时长 ≈ 2 分钟

功能说明

在Windows资源管理器中通过右键菜单直接调用PowerShell的Get-FileHash命令，支持8种哈希算法验证：

  • SHA1
  • SHA256
  • SHA384
  • SHA512
  • MACTripleDES
  • MD5
  • RIPEMD160
  • 全选验证模式（同时显示所有算法结果）
