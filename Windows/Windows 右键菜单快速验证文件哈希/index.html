Windows 右键菜单快速验证文件哈希

发表于 2025-05-10 分类于 Windows 阅读次数： 本文字数： 640 阅读时长 ≈ 2 分钟

功能说明

在Windows资源管理器中通过右键菜单直接调用PowerShell的Get-FileHash命令，支持8种哈希算法验证：

  • SHA1
  • SHA256
  • SHA384
  • SHA512
  • MACTripleDES
  • MD5
  • RIPEMD160
  • 全选验证模式（同时显示所有算法结果）

结果解读示例

各字段含义：

  • Algorithm：使用的哈希算法
  • Hash：计算得到的哈希值（HEX格式）
  • Path：被验证文件的完整路径

使用指南

基本操作流程

  1. 在资源管理器中右键点击需要验证的文件
  2. 选择 Verify Hash 菜单项
  3. 从8种算法中选择需要的验证方式：
    • 单个算法：选择对应算法名称（如 SHA256）
    • 全算法验证：选择 Verify all
