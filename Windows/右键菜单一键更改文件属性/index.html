这样的操作还不够快，那有什么方法可以更加方便的隐藏文件呢？于是我在右键菜单上增加了个小功能，可以快捷更改文件的一些常用属性，对于文件夹也同样适用，效果如下

原理

Windows 中可以用命令 ‘attrib’ 更改文件属性，它可以显示、设置或删除分配给文件或目录的属性，语法如下

参数如下

参数 说明
+r 设置 (+) 或清除 (-) 只读文件属性
+a 设置 (+) 或清除 (-) 存档文件属性
+s 设置 (+) 或清除 (-) 系统文件属性
+h 设置 (+) 或清除 (-) 隐藏文件属性
+i 设置 (+) 或清除 (-) 非内容索引文件属性
/s 将选项应用到当前目录中及其所有子目录中的匹配文件
/d 将选项应用到目录
/l 将选项应用到符号链接，而不是符号链接的目标

更多详细参数请查看 Microsoft 文档
