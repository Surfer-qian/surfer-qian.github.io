前言

为 Hexo 文章设置一个链接 “阅读全文” 来进入文章的详情页，同时只显示前言部分——就是你现在看到的样子。

点击阅读全文看看是如何实现的

使用 手动截断

在需要截断的位置添加 <!--more--> ，之后的内容只有在点击文章详情页才能看到，并且 <!--more--> 之前的内容在文章详情里也能一并看到

利用文章的 front-matter

在文章的 front-matter 添加 description ，并添加描述，此描述只在首页显示，在文章详情页不会显示
